How to Start A Bullet Journal

1.Log In

Create an account or log in if you already have one.

2.Home Page

Log in takes you to the home page where the library of all your previous journals are kept

3.Journal

Click the Add New Journal Button at the top of the home page to start a new journal

4.Canvas

Once in the canvas use the pen tools to start drawing.

5.Design

Use the template button to populate a template to start on or create your own from scratch

6.Save

Once you're done, click the save button to save your journal. The exit button takes you back to the home page
